#wup is a cli-journal

##commands
- win : input post
- wed : edit
  - ex. wed <key> or 
  - echo <key> | wed
- wgrep : grep
  - wgrep <q> or 
  - echo <q> | wgrep
- wkeys : list keys
- wls : list 
  - ex. wls <any>
- wgrepz :  fzf query
- wrm : rm 
  - ex. wrm <key> or 
  - echo <key> | wrm
- wcat : cat 
  - ex. wcat <key> or 
  - echo <q> | wview
- wcatz : fzf view
