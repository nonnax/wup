#wup is a cli-journal

##commands
- win : input post
- wed : edit
  - ex. wed <key> or echo <key> | wed
- wgrep 
  - wgrep <q> or echo <q> | wgrep
- wkeys : list keys
- wls : list 
  - ex. wls <any>
- wq :  fzf query
- wrm : rm 
  - ex. wrm <key> or echo <key> | wrm
- wview : view 
  - ex. wview <key> or echo <q> | wview
- wviewz : fzf view
